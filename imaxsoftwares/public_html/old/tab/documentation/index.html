HTML Structure

Template grid system is based on a 1170px width. Below 1230px wide, the grid system will become fluid and stack vertically.

For responsiveness the grid will support a tablet layout and a mobile layout

  • Between 1230px and 690px wide, the tablet layout will be enabled.
  • Below 690px wide, the mobile layout will be enabled.

Buttons

There are 4 types of buttons: Small, Regular, Large and outline. Each button can have unlimited background, font and border colors for basic style and hover style

Down below is a list of meaning of certain class prefixes:

  • color-* -> This class syntax is configured for font color styles.
  • bg-* -> This class syntax is configured for background color styles.
  • border-* -> This class syntax is configured for border color styles.
  • hover-color-* -> This class syntax is configured for font color styles on hover.
  • hover-bg-* -> This class syntax is configured for background color styles on hover.
  • hover-border-* -> This class syntax is configured for border color styles on hover.

Note: All the colors used in the template is written in css/colors.css file.

Counters

There are 4 types of counters available in this template: Charts, Progress Bars, Numbers, Countdowns

- Charts: To add the percentage you want, just change the "data-animate-percent" attribute value to your needs

- Progress bars: To add the percentage you want, just change the element with "percentage" class text and the element with "bar" class "data-percentage" attribute value to your needs

- Numbers: To add the percentage you want, just change the element with "number" class value to your needs

- Countdowns: To add the date you want, just change the data attributes to your needs.
